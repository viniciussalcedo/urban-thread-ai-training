# Urban Thread AI Training

A demonstration interface for AI-assisted stock analysis training.

## Live Demo
[https://viniciussalcedo.github.io/urban-thread-ai-training/](https://viniciussalcedo.github.io/urban-thread-ai-training/)

## Overview
This dashboard provides a frontend interface for reviewing AI-generated stock analyses and capturing feedback. It demonstrates a human-in-the-loop training workflow.

## Features
- Stock analysis simulation
- Feedback capture interface
- Progress tracking
- Data export

## Usage
1. Enter a ticker symbol (e.g., NVDA, AAPL, TSLA)
2. Click "Run Rex Analysis" to see simulated AI analysis
3. Review the analysis and provide corrections if needed
4. Corrections are saved locally in your browser

## Technical Notes
- Frontend: HTML/CSS/JavaScript
- Data: LocalStorage for browser persistence
- Integration: Demonstrates connection to analysis backend

## Privacy
All data remains in your browser. No information is sent to external servers.
